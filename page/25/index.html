vw和vh单位的使用

发表于 2019-12-04 更新于 2022-12-27 分类于 css
.vh {
  width: 50vw; //元素宽度为视窗宽度的 50%
  height: 80vh; //元素高度为视窗高度的 80%
}

相对于视口的宽度，视口被均分为 100 单位的 vw
相对于视口的高度。视口被均分为 100 单位的 vh
vw 和 vh 可以理解为相对于视窗宽高的百分比单位而不是相对于父元素
