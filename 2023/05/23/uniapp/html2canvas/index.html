html2canvas 将页面转为图片并保存到本地相册

发表于 2023-05-23 更新于 2023-08-04 分类于 uni-app 阅读次数：

技术点

  • html2canvas：将网页绘制 base64 的图片
  • plus.nativeObj.Bitmap：下载 base64 的 png 图片，临时存放起来
  • uni.saveImageToPhotosAlbum：将临时存放的 png 图片，存放到手机相册
  • plus.io.resolveLocalFileSystemURL：删除临时存放的图片

运用

安装 html2canvas

RenderJs

因为 html2canvas 要进行 dom 操作，所以我们得使用 renderjs。script 就要分为两部分，一个是纯的 script 标签块，一个是带有 module、lang 的 script 标签块
