参数说明类型默认值

minHeight为了避免列表数据过少，下拉刷新时列表区域被遮挡String, Number0

Slots 插槽

名称说明
内容填充
content自定义列表每一项
列表相关

success下拉成功提示

Events

1. 当页面有筛选条件或者 tab 切换需要更新列表数据时，需要主动调用 reset 方法重置数据

2. 父组件需要获取数据长度等相关的属性时，可调用 setData 获取列表数据

3. 当接口请求发生错误时，父组件可调用 error 捕捉错误信息
