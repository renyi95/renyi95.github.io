怎么使用 git 将多次提交压缩成一次提交？

发表于 2022-07-30 更新于 2023-08-04 分类于 git 阅读次数：

在使用 Git 作为版本控制的时候，我们可能会由于各种各样的原因提交了许多临时的 commit，而这些 commit 拼接起来才是完整的任务。那么我们为了避免太多的 commit 而造成版本控制的混乱，通常我们推荐将这些 commit 合并成一个。
